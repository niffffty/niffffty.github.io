电路笔记1

一、电路分析法

  • 网孔电流法（回路电流法）

    • 解题步骤：

      • 画绕行方向

      • 列电流方程：R自I自+R互I互=电压升

      • 解方程

    • 题型：

      • 只有电压源：按步骤

      • 有电流源：先设电流源电压，再补电流方程

      • 有受控源：设受控源电压，补方程

  • 节点电压法

    • 解题步骤：

      • 标节点，标接地点

      • 列节点电压方程：G自U自-G互U互=流进电流

      • 解方程

    • 题型：

      • 只有电流源

      • 含电压源：将电压源等效为电流源，设电流

      • 含受控源：补受控源方程

  • 支路电流法

    • 步骤：KVL，KCL，解方程

二、电路定理

叠加定理

  • 步骤：

    • 去掉电流源，电压源独立作用，KVL/KCL求U1，i1

    • 去掉电压源，电流源独立作用，求U2，i2

    • 叠加：U=U1+U2

戴维宁定理

  • 步骤：

    • 拿走待求支路，求Uoc

    • 独立源置零，求等效电阻Req

    • 画出戴维宁等效电路（一个电阻，一个电压源）

    • 放回待求支路

  • 题型：

    • 无受控源：按步骤解题

    • 有受控源：第二步求Req时，外加电压

最大功率传输

  • 步骤

    • 用戴维宁定理求等效电路

    • 用公式：Pmax=Uoc方/4Req

电路响应

  • 换路定理：电路发生变化（开关闭合）时，电容电压不变，电感电流不变

  • 解题步骤：

    • 画t=0-的电路（电容开路，电感短路），求Uc，iL

    • 换路定理求0+时刻电容/电感的电压/电流

    • 画出t=0+的电路（电容当电压源，电感当电流源），KCL/KVL求解方程

三要素法求响应

  • 概念
    • 零输入响应：换路后没有电源
    • 零状态响应：环路前U，i等于0的电路
    • 全响应：三要素公式求的
  • 解题步骤：
    • 前三步不变
    • 求等效电阻Ri，求时间常数 RiC，L/Ri
    • 画t=∞时的电路，求f(∞)，此时是稳态，电容开路/电感短路
    • 三要素法公式求U/i
